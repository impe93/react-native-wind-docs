## Customization

:::info

To get an introduction on how customization works, it is recommended to visit the [dedicated page in the documentation](/docs/core/customization)

:::

To customize the classes regarding the background colors, you go to modify the fields under the property called `colors`.

```js [title="wind.config.js"]
import { customize } from "react-native-wind";

customize({
  theme: {
    colors: {
      primary: "#333333",
      secondary: {
        light: "#f3f3f3", // Light shade
        dark: "#212121", // Dark shade
      },
    },
  },
});
```

As show in the example block, there are 2 ways to customize colors:

1. Immediatly write the color as the value
2. Specify an object with shades of that color

The library will automatically generate classes for every other utility that use colors, in the background color case as:

1. The first option with just the color value generate the classes:

- `border-primary`
- `border-t-primary`
- `border-b-primary`
- `border-r-primary`
- `border-l-primary`
- `border-e-primary`
- `border-s-primary`

2. The second option with the specified shades generate the classes:

- `border-secondary-light`
- `border-t-secondary-light`
- `border-b-secondary-light`
- `border-r-secondary-light`
- `border-l-secondary-light`
- `border-e-secondary-light`
- `border-s-secondary-light`
- `border-secondary-dark`
- `border-t-secondary-dark`
- `border-b-secondary-dark`
- `border-r-secondary-dark`
- `border-l-secondary-dark`
- `border-e-secondary-dark`
- `border-s-secondary-dark`
- `bg-secondary-light` `bg-secondary-dark`
